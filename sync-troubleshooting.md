# 🔄 同期問題トラブルシューティングガイド

## 問題の概要
WEBとモバイルでデータが同期しない問題の解決方法を説明します。

## 🔍 原因の特定

### 1. ブラウザキャッシュの問題
**症状:** 古いバージョンのアプリが動作している
**解決方法:**
- `Ctrl + Shift + R` (Windows/Linux) または `Cmd + Shift + R` (Mac) でハードリフレッシュ
- シークレットモード/プライベートブラウジングで試行
- ブラウザのキャッシュを完全にクリア

### 2. ネットワーク接続の問題
**症状:** データの保存・読み込みが失敗する
**解決方法:**
- 安定したインターネット接続を確保
- モバイルデータとWi-Fiを切り替えて試行
- VPNを使用している場合は一時的に無効化

### 3. Firebase認証の問題
**症状:** ログインは成功するがデータが同期しない
**解決方法:**
- ログアウトして再ログイン
- 別のブラウザで試行
- Firebase Consoleでプロジェクト設定を確認

### 4. Firestore接続の問題
**症状:** データベースエラーが発生する
**解決方法:**
- Firebase ConsoleでFirestore Databaseが有効になっているか確認
- セキュリティルールを確認
- プロジェクトの課金状況を確認

## 🛠️ デバッグツールの使用方法

### 同期問題デバッグツール
`sync-debug.html` を使用して問題を特定できます：

1. **デバイス情報確認**
   - ブラウザ、プラットフォーム、画面サイズを確認
   - デバイス間の違いを特定

2. **認証状態確認**
   - 現在のユーザー認証状態を確認
   - ユーザーIDとメールアドレスを確認

3. **Firestore接続確認**
   - データベースへの接続状況を確認
   - エラーメッセージを詳細に表示

4. **リアルタイムリスナー確認**
   - リアルタイム同期の動作状況を確認
   - データ更新の受信状況を監視

5. **データ読み込みテスト**
   - 現在のデータを読み込み
   - データの内容と件数を確認

6. **データ保存テスト**
   - テストデータを保存
   - 保存処理の成功/失敗を確認

7. **同期テスト**
   - 複数デバイス間の同期状況を確認
   - 最終更新時刻とデバイス情報を比較

8. **ネットワーク状態確認**
   - オンライン/オフライン状態を確認
   - 接続速度と品質を確認

## 📱 モバイルでの同期確認

### iOS Safari
1. 設定 > Safari > プライベートブラウジングを無効化
2. 設定 > Safari > ウェブサイトのデータを消去
3. Safariを再起動してアプリにアクセス

### Android Chrome
1. Chrome > 設定 > プライバシーとセキュリティ
2. 閲覧履歴データを消去
3. Chromeを再起動してアプリにアクセス

## 🔧 技術的な解決方法

### 1. キャッシュバスティング
アプリのバージョン管理により、古いキャッシュを強制的に更新：
```javascript
const appVersion = '1.0.1'; // 同期問題修正版
```

### 2. リアルタイム同期の改善
データの整合性チェックを追加：
```javascript
// データが実際に変更された場合のみ更新
const routinesChanged = JSON.stringify(routines) !== JSON.stringify(newRoutines);
const completionsChanged = JSON.stringify(completions) !== JSON.stringify(newCompletions);
```

### 3. オフライン対応
ネットワーク状態を監視し、接続復旧時に自動同期：
```javascript
window.addEventListener('online', () => {
    // 接続復旧時にデータを再同期
    loadUserData();
});
```

### 4. リトライ機能
保存失敗時の自動再試行（最大3回）：
```javascript
if (window.saveRetryCount < 3) {
    window.saveRetryCount++;
    setTimeout(() => {
        saveData();
    }, 2000 * window.saveRetryCount); // 指数バックオフ
}
```

## 📊 同期状況の確認方法

### 1. コンソールログの確認
ブラウザの開発者ツールで以下を確認：
- データ読み込みログ
- データ保存ログ
- リアルタイム更新ログ
- エラーメッセージ

### 2. データの比較
各デバイスで以下を確認：
- ルーティンの件数
- 完了記録の件数
- 最終更新時刻
- デバイス情報

### 3. ネットワークタブの確認
開発者ツールのNetworkタブで：
- Firebase APIへのリクエスト状況
- レスポンスの成功/失敗
- リクエストのタイミング

## 🚨 緊急時の対処法

### 1. データが完全に同期しない場合
1. すべてのデバイスでログアウト
2. ブラウザキャッシュをクリア
3. メインデバイスでログイン
4. データを確認・修正
5. 他のデバイスで順次ログイン

### 2. データが消失した場合
1. Firebase Consoleでデータを確認
2. バックアップがあれば復元
3. 必要に応じてデータを再入力

### 3. 認証ができない場合
1. パスワードをリセット
2. 別のブラウザで試行
3. シークレットモードで試行
